# Recipe Finder App 🍳

A React-based web application that allows users to search recipes by ingredient using TheMealDB API.

## 🚀 Features
- Search recipes by ingredient
- Display recipes in responsive cards
- API integration with TheMealDB
- Loading spinner for better UX
- Clickable recipe cards (detail view in progress)

## 🛠 Tech Stack
- React (Vite)
- Material UI
- JavaScript (ES6+)
- REST API

## 📦 Installation

```bash
npm install
npm run dev
